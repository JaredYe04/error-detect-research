Given an FSF specification, HSP-Agile parses SpecIR regions, generates an LLM
candidate, and iterates repair up to $K{=}3$ attempts.
Each failed attempt emits Semantic Feedback IR: scenario label, violated guard,
witness input, and optional pattern-guard hits.

Accept is conjunctive:
$\mathrm{Accept}(c,t) \Leftrightarrow (\forall w.\,\mathrm{Conf}_w(c,t){=}1)
\wedge \neg\exists p.\,\mathrm{severity}(p){\geq}\tau$.
Mode~M applies hard pattern screening before Accept; B2 uses unstructured
test pass/fail only.

The Semantic Feedback IR is the repair hinge: variant~C (full IR) reaches
86.9\% mean conformance vs.\ 79.1\% for test-only variant~A on the same 120 tasks
(E6)---isolating specification structure from prompt volume (E14 confirms under
length-matched sweeps).
