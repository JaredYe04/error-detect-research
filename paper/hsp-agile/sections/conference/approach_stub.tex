Given an FSF specification, HSP-Agile parses \textbf{SpecIR} regions
$R_i = G_i \wedge \bigwedge_{j<i}\neg G_j$, synthesises an LLM candidate, and
repairs under mode~M.
Failed attempts emit a typed \textbf{Semantic Feedback IR}: scenario label,
violated guard, SMT witness, expected postcondition, and optional pattern
hits---rendered into the repair prompt (no free-form verbal memory).

Release is conjunctive ($\tau{=}1$, $P^H(c)$ high-severity pattern count):
\begin{equation}
  \mathrm{Accept}(c,t)
  \;\iff\;
  \mathrm{Conf}(c,t)\geq\tau
  \;\wedge\;
  P^H(c)=0.
  \label{eq:accept-conf}
\end{equation}
Mode~M applies critical-only hard pattern screening before Accept; B2 uses
unstructured test pass/fail only.
